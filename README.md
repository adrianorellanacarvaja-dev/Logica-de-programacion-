# Logica-de-programacion-

SOFTWARE GENERADOR DE CONTRASEÑAS SEGURAS
 
Descripción:
Este proyecto consiste en el desarrollo de un programa en Python capaz de generar contraseñas seguras y personalizables. El sistema permite al usuario definir la longitud y los tipos de caracteres que desea incluir, garantizando así mayor seguridad en la creación de credenciales digitales.



Objetivo:
-Desarrollar una herramienta que:
-Genere contraseñas aleatorias.
-Permita configurar la longitud.
-Incluya letras mayúsculas, minúsculas, números y símbolos.
-Refuerce buenas prácticas de seguridad informática.



Requisitos:
-Python 3.x instalado
-Librerías estándar de Python:
-random
-string



Cómo ejecutar el programa:
1.-Descargar o clonar el repositorio.
2.-Abrir una terminal en la carpeta del proyecto.
3.-Ejecutar el archivo principal
4.-Ingresar los parámetros solicitados (longitud y tipos de caracteres).
5.-El sistema mostrará la contraseña generada en pantalla.
6.-Funcionamiento del programa



El programa:
1.-Solicita al usuario la longitud de la contraseña.
2.-Permite elegir si desea incluir:
-Letras mayúsculas
-Letras minúsculas
-Números
-Símbolos
3.-Construye una lista de caracteres permitidos.
4.-Genera una contraseña aleatoria utilizando selección aleatoria.
5.-Muestra el resultado final.



Pruebas realizadas:
-Pruebas con diferentes longitudes.
-Validación de combinaciones de caracteres.
-Verificación de que no se generen contraseñas vacías.
-Manejo básico de errores en caso de entradas inválidas.



Resultados:
-El programa cumple con los requisitos de generación segura, 
permitiendo personalización y garantizando aleatoriedad en cada ejecución.
